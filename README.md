# RSSchool NodeJS websocket task template

## Installation

1. Clone/download repo
2. `cd rss-nodejs`
3. `git checkout battleship`
4. `npm install`

## Usage

**Development**

`npm run start:dev`

- App served @ `http://localhost:8181` with nodemon

**Production**

`npm run start:prod`

- App served @ `http://localhost:8181` without nodemon
